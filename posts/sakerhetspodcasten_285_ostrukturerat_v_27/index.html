Azure: nya säkrare default, ingen egress ut

00:46:30 Azure stänger egress / utgående access i September. Har du inte konfigurerat access, så når din VM inte ut!

After September 30, 2025, new virtual networks will default to requiring explicit outbound connectivity methods instead of having a fallback to default outbound access connectivity.

All virtual machines that require public endpoint access will need to use explicit outbound connectivity methods such as Azure NAT Gateway, Azure Load Balancer outbound rules, or a directly attached Azure public IP address.
